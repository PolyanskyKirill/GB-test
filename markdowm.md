## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить звездочкой (*). 
Например вот так: 
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерование списки, необходимо пункты просто пронумеровать. Например, вот так:

1. Первый элементЫ
2. Второй элементЫ
3. Третий элемент
